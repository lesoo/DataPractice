  • Panel Data Analysis

  • pip install pandas

Data Frame¶

  • 데이터가 틀에 맞춰서 저장된 형태
  • 가로 x 세로 직사각형 모양으로 틀을 갖춰 저장
  • 엑셀, 테이블, 데이터 프레임 등 분야마다 다르게 불림
  • 한 row 안의 데이터는 Dictionary처럼 저장
  • 한 column 안의 데이터는 list처럼 저장
  • 한 데이터 프레임은 row들이 list처럼 저장

series¶

  • 데이터가 순차적으로 나열된 1차원 배열 형태
  • 키와 값이 '{k:v}' 형태로 짝을 이루는 파이썬 Dict와 비슷한 구조를 갖음
  • k --> index
  • v --> value
